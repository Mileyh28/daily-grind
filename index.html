The assignment involves creating a dynamic daily special page for a client, showcasing a different coffee special for each day of the week. Using HTML, CSS, and JavaScript, the page will change its content based on the current day. Each day will feature:

  • A unique image of the coffee, with an appropriate alt tag.
  • A description of the coffee, including the name of the day.
  • A unique color that complements the coffee image and description.
  • A link to view the special for each day.

The content will be rotated using the Date object to vary images or content based on the day of the week. This involves using conditionals (like 'if' statements or switch blocks) to determine and display the appropriate HTML content for each day. The goal is to implement all these features within a single page.
